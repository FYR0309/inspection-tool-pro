# UI 重设计 — 去 AI 味，工业风

> 目标：让检查工具 Pro 看起来像专业的工厂信息化系统，而非花哨的互联网产品
> 用户：30-50 岁男性车间主任/安全员，手机端（iPhone Safari/微信）
> 行业：泛工业通用（制造业/化工/建筑/仓储/电力等）

---

## 1. 配色方案

暖色调全面替换为冷灰+深蓝体系。仅保留工厂橙做警示色。

| 角色 | 旧值 | 新值 |
|------|------|------|
| --primary | `#c0833c` | `#3b5998` |
| --bg | `#f3f0eb` | `#eef0f3` |
| --card | `#ffffff` | `#f7f8fa` |
| --success | `#3d7256` | `#4a8c5c` |
| --danger | `#c4553d` | `#d14343` |
| --warning | `#d4952b` | `#e07b20` |
| --text | `#2d2822` | `#2c3038` |
| --text-secondary | `#7a7368` | `#6b7280` |
| --border | `#e0dbd2` | `#dfe1e5` |
| 新增 --shadow | 无 | `none`（去掉全部阴影） |

## 2. 字体与排版

信息密度提升，字小一号、间距收紧。

| 元素 | 旧值 | 新值 |
|------|------|------|
| 基础字号 | 15px | 14px |
| 标题字号 | 22px | 18px（字间距 -0.02em） |
| 辅助字号 | 13px | 12px（opacity 0.65） |
| 卡片间距 | 10px | 6px |
| 页面内边距 | 16px | 12px |
| 按钮最小高度 | 44px | 40px |
| 行高 | 1.6 | 1.5 |

## 3. 去掉 emoji

规则：所有界面文字中的 emoji 移除，仅照片槽位的 📷 保留（缩小尺寸）。

| 区域 | 旧文字 | 新文字 |
|------|--------|--------|
| 首页副标题 | 💡 支持拍照、语音输入、AI润色 | 删除整行 |
| 导入提示 | 💡 点击卡片上的 📥 可导入... | 删除整行 |
| 设置栏（已填） | 🏢 公司 · 👤 部门 · 📅 日期 · ⚡ 免费版 | 纯文字，无 emoji |
| 设置栏（未填） | ⚠️ 点击设置公司名称... | 改为红色文字 "请设置公司名称" |
| 模板分类 | —— 📦 内置模板 —— / —— 📥 我的模板 —— | "内置模板" / "我的模板" |
| 按钮 | ✨ AI润色、📥 下载 Word、📤 导出... | AI润色、下载 Word、导出... |
| 卡类型标签 | 🏭 📄 | 4px 色块 + 文字 |
| 激活状态 | 🔓 Pro版 / ⚡ 免费版 | 色块标签 "PRO" / "免费版" |

## 4. 按钮风格

方正、清晰、无多余装饰。

| 属性 | 旧值 | 新值 |
|------|------|------|
| 圆角 | 8px | 4px |
| 阴影 | 主按钮有彩色阴影 | 无阴影 |
| 主按钮 | background + box-shadow | 仅 background |
| 次按钮 | 彩色边框 | 灰色边框 `#d0d3d8` |
| 文字按钮 | 带 emoji | 纯文字（如不需要按钮形态则用下划线链接） |
| 危险操作 | emoji+文字红色 | 文字红色，无 emoji |
| 分组间距 | 10px | 8px |

## 5. 卡片与列表

| 元素 | 旧值 | 新值 |
|------|------|------|
| 模板卡片左边框 | 彩色 4px | 主色 4px，无渐变 |
| 卡片阴影 | `0 1px 4px rgba(...)` | 无 |
| 卡片圆角 | 8px | 2px |
| 条目列表分割 | 圆角卡片+阴影 | 细线 border-bottom |
| 图片缩略图圆角 | 8px | 2px |
| 卡片选中态 | 主色光晕 | 1px 主色边框 |

## 6. 照片槽位

| 属性 | 旧值 | 新值 |
|------|------|------|
| 空槽动画 | 呼吸动画 | 无动画，静态 |
| 空槽边框 | 2px dashed | 1px dashed `#d0d3d8` |
| 有照片边框 | 渐变彩色 | 1px solid `#dfe1e5` |
| 修图按钮 | 紫色渐变 `#7b6db5` | 灰色 `#6b7280` |

## 7. 主页信息架构

| 改动 | 说明 |
|------|------|
| 删除顶部副标题 | "选择一个模板开始，或导入您的 Word 模板" 去掉 |
| 删除提示行 | 首页两条 💡 提示文字均删除 |
| 设置栏下沉 | 从标题下方移到页面底部（草稿箱上方），缩小字号 |
| 模板分组标题简化 | 去掉 emoji 和装饰线 |

## 8. 实施范围

涉及文件：
- `css/style.css` — CSS 变量重定义 + 组件样式重写
- `js/ui.js` — 所有 HTML 模板中的 emoji 删除 + 文字调整
- `templates/safety.json`, `templates/5s.json` — industry icon 映射改为文字标签

不变文件：
- `js/app.js`, `js/ai.js`, `js/db.js`, `js/docx-gen.js`, `js/docx-parser.js`, `js/docx-template-cloner.js`, `js/camera-voice.js`, `js/importer.js`, `js/activate.js`, `js/config.js`, `js/utils.js`
